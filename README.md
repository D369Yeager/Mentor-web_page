# 🎓 Alumni Mentorship Platform

A simple and responsive **Alumni Mentorship Platform** built using **HTML, CSS, and JavaScript**. This project provides a basic interface for connecting students with alumni mentors, allowing mentorship booking requests, community discussions, and an overview dashboard.

---

## 📌 Features

### 👨‍🏫 Mentor Profiles

* Displays mentor information including:

  * Name
  * Domain/Expertise
  * Experience
  * Short Bio
  * Availability

### 📅 Booking Requests

* Students can request mentorship sessions.
* Select a mentor from the available list.
* Choose a preferred session date.
* Booking requests are stored using **LocalStorage**.

### 💬 Open Discussion Forum

* Students and alumni can:

  * Post questions
  * Share insights
  * Participate in discussions
* Forum posts are saved in **LocalStorage**.

### 📊 Dashboard

Provides a quick overview of:

* Total number of mentors
* Total booking requests
* Total discussion posts
* List of all booking requests

## 💡 How It Works

1. Browse available mentor profiles.
2. Select a mentor and submit a booking request.
3. Participate in the discussion forum by posting questions or insights.
4. View statistics and booking information on the dashboard.
5. All bookings and discussion posts remain available after refreshing the page through browser LocalStorage.
